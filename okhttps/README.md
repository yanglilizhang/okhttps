# OkHttps

#### 介绍

基于 OkHttps 核心模块


##### Maven

```xml
<dependency>
     <groupId>cn.zhxu</groupId>
     <artifactId>okhttps</artifactId>
     <version>3.5.3</version>
</dependency>
```

##### Gradle

```groovy
implementation 'cn.zhxu:okhttps:3.5.3'
```
