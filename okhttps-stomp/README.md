# OkHttps Stomp

#### 介绍

基于 OkHttps 的 Stomp 协议实现


##### Maven

```xml
<dependency>
     <groupId>cn.zhxu</groupId>
     <artifactId>okhttps-stomp</artifactId>
     <version>3.5.3</version>
</dependency>
```

##### Gradle

```groovy
implementation 'cn.zhxu:okhttps-stomp:3.5.3'
```
