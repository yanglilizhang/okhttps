# OkHttps Gson

#### 介绍

基于 OkHttps 与 Gson 集成扩展


##### Maven

```xml
<dependency>
     <groupId>cn.zhxu</groupId>
     <artifactId>okhttps-gson</artifactId>
     <version>3.5.3</version>
</dependency>
```

##### Gradle

```groovy
implementation 'cn.zhxu:okhttps-gson:3.5.3'
```
